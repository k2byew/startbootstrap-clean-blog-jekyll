Navigating with the keyboard can be quite different when going from PC to Mac:

Home - Ctrl + a

End - Ctrl + e

Previous Word - Alt + Left Arrow

Next Word - Alt + Right Arrow

New Tab - CMD + t

New Window - CMD + n
